# PRO калькулятор ротанга

Статический сайт (без сборщиков): лендинг сверху и калькулятор ниже.

## Как запустить
Открой файл `index.html` в браузере.

## Структура
- `index.html` — страница (hero + калькулятор)
- `css/styles.css` — стили и светлая/тёмная темы
- `js/app.js` — логика (вкладки, переключение темы, расчёт)

## Примечание
Исходная версия оставлена как `2.html` (на всякий случай).
